Project Title

Contact Book Management System



Project Description

The **Contact Book Management System** is a simple, menu-driven Python console application designed to store and manage contact details efficiently.
It allows users to **add, view, search, and save contact records** using basic Python programming concepts.
This project is beginner-friendly and follows the standard guidelines of a Python mini project.



Objective

* To understand how contact information can be managed using Python
* To practice core Python concepts such as lists, dictionaries, functions, loops, and file handling
* To build a simple, interactive, menu-driven console application



Features

* Add new contact records
* Display all contact details
* Search contacts by name
* Save contact records to a text file
* Menu-driven and user-friendly interface
* Uses pre-defined sample contact data


Concepts Used

* Python Basics
* Lists and Dictionaries
* Functions
* Conditional Statements
* Loops
* Exception Handling (`try` & `except`)
* File Handling (writing data to a `.txt` file)



Instructions to Run the Program

1. Make sure **Python 3.x** is installed on your system
2. Download or copy the project file (e.g., `contact_book.py`)
3. Open Command Prompt / Terminal
4. Navigate to the project folder
5. Run the program using:


python contact_book.py




Input Guidelines

* Contact ID should be a **number**
* Name should contain **alphabetic characters only**
* Phone number should be **numeric**
* Email ID should be entered in a **valid format**
* Follow the on-screen menu instructions carefully



File Handling Details

* Contact data is stored using Python lists and dictionaries
* All contact records are saved in a file named **`contacts.txt`**
* Each contact is written to the file as a dictionary
* The file is created automatically if it does not exist



Sample Data Fields

* Contact ID
* Contact Name
* Phone Number
* Email ID
